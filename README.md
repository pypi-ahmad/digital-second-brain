# digital-second-brain

## Overview

Digital Second Brain is a local-first handwritten-notes workflow with a Gradio interface.

## Tech Stack

- Python (requirements.txt based)

## Getting Started

### Prerequisites

- Git
- Runtime dependencies for this project's stack

### Installation

```bash
uv venv
uv pip install -r requirements.txt
```

## Usage

Run the primary app with `uv run app.py`.

## Testing

Run tests with `uv run pytest` from repository root.
